# FORUS 1C Practice
## О проекте

Во время прохождения производственной практики в компании «Форус» выполнялась работа с информационной базой 1С для магазина.

Практика была направлена на знакомство с принципами работы информационной системы, структурой базы данных и выполнением типовых задач по сопровождению системы.

---

## Выполненные задачи

- разработка информационной базой 1С;
- загрузка и выгрузка данных;
- работа со справочниками;
- работа с документами;
- проверка корректности данных;
- изучение структуры базы данных;
- выполнение практических заданий в рамках производственной практики.

- 1С:Предприятие
- Информационная база 1С
